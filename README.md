
# AdventureWorks Internet Sales - Power BI Dashboard

## Overview
This project presents a fully interactive **Power BI** dashboard built using the **AdventureWorks Internet Sales** dataset.  
It provides insights into global revenue performance, product sales, customer behaviour, country-level comparisons, and logistics efficiency.  
Use the slicers for **Year**, **Quarter**, **Country**, and **Product** to explore the data dynamically.

---

##  Key Features

###  Executive KPIs
Top‑level metrics surfaced on the main Internet Sales page:
- **Revenue**
- **Profit**
- **Profit Margin (%)**
- **Orders**
- **Customers**
- **Profit per Order**
- **Average Order Value**
- **Cost of Goods Sold (COGS)**

###  Product Insights
- **Top 10 Products by Revenue** (e.g., Mountain, Road, Touring Bikes)
- Category/product performance comparisons
